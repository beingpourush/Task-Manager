# 📝 Task Manager

A simple **Task Manager Web App** built using **HTML, CSS, and JavaScript**.
It helps users manage daily tasks efficiently with a clean and minimal interface.

---

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Simple and user-friendly UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
taskmanager/
│── taskmanager.html
│── style.css
│── script.js
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the file in your browser:

```
taskmanager.html
```
